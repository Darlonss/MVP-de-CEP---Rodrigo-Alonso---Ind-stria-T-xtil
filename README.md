# **MVP — Controle Estatístico de Processo: Previsão de Produtividade em Linha de Confecção de Vestuário (Rodrigo Alonso)**

## **Sobre o Projeto**

Este repositório contém o MVP desenvolvido para a disciplina de Controle Estatístico de Processos (ERP) da Universidade de Brasília, sob orientação do Professor Dr. André Luiz Marques Serrano.

O projeto aplica técnicas de **Machine Learning** e **Controle Estatístico de Processo (CEP)** sobre o dataset público *Garment Worker Productivity* (UCI Machine Learning Repository), com o objetivo de prever se equipes de produção de uma fábrica de confecção irão atingir sua meta de produtividade diária.

## **Problema Abordado**

A indústria de confecção é intensiva em mão de obra e altamente sensível a variações de desempenho no chão de fábrica. A partir de dados reais de 1.197 registros de equipes dos departamentos de costura e acabamento, foi construído um modelo capaz de identificar, com base em variáveis operacionais, quais equipes têm maior ou menor probabilidade de cumprir a meta estabelecida pela gestão.

## **Dataset**

**Productivity Prediction of Garment Employees** — Kaggle
